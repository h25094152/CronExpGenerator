## 基于JavaFX 的 Quartz Cron 表达式生成器

通过简单的设置快速生成 Quartz Cron 表达式

#### 编译需求
- Java 1.8

#### 主要功能
1.  简单的选择快速生成Cron表达式
2.  解析cron表达式预计执行时间
